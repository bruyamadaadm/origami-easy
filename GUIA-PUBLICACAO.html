🦢 Guia de Publicação - Origami Easy PWA

Este guia te leva do zero até ter seu app publicado online, instalável no celular e computador, em menos de 10 minutos.

O que você vai ter no final:

  • Um link público (ex: origami-easy.vercel.app) que qualquer pessoa pode acessar
  • App instalável no celular (Android/iPhone) e computador
  • Funciona 100% offline depois de instalado
  • Dados salvos localmente no dispositivo do usuário

Passo 1: Criar conta no GitHub

Se já tem conta, pule para o Passo 2

  1. Acesse github.com/signup
  2. Crie sua conta com email e senha
  3. Confirme o email

Passo 2: Criar repositório no GitHub

Subir os arquivos do app

  1. No GitHub, clique no + no canto superior direito → "New repository"
  2. Nome: origami-easy
  3. Marque "Public" (necessário para Vercel gratuita)
  4. Clique "Create repository"
  5. Na página que abrir, clique "uploading an existing file"
  6. Arraste TODOS os arquivos da pasta origami-easy-pwa:
    • index.html
    • manifest.json
    • sw.js
    • vercel.json
    • Pasta icons/ (arraste a pasta inteira)
  7. Clique "Commit changes"
⚠️ Importante: Arraste os arquivos diretamente, NÃO a pasta origami-easy-pwa. Os arquivos devem ficar na raiz do repositório.

Passo 3: Criar conta na Vercel

Hospedagem gratuita

  1. Acesse vercel.com/signup
  2. Clique em "Continue with GitHub"
  3. Autorize a conexão com seu GitHub

Passo 4: Publicar na Vercel

Deploy automático

  1. Na Vercel, clique "Add New..." → "Project"
  2. Encontre o repositório origami-easy e clique "Import"
  3. Na tela de configuração:
    • Framework Preset: Other
    • Root Directory: deixe vazio (ou ./)
    • Build Command: deixe vazio
    • Output Directory: ./
  4. Clique "Deploy"
  5. Aguarde ~30 segundos...
✅ Pronto! Seu app estará disponível em algo como:
https://origami-easy.vercel.app

Você pode compartilhar esse link com qualquer pessoa!

Passo 5: Personalizar o domínio (opcional)

Trocar o nome do link

  1. Na Vercel, vá em Settings → Domains
  2. Você pode mudar o subdomínio gratuito (ex: meu-financeiro.vercel.app)
  3. Ou conectar um domínio próprio se comprar um (ex: origamieasy.com.br ~R$40/ano)

Como instalar o app no celular/computador

Depois de publicado:

  1. No celular (Android): Abra o link no Chrome → menu ⋮ → "Instalar app" ou "Adicionar à tela inicial"
  2. No celular (iPhone): Abra no Safari → botão compartilhar → "Adicionar à Tela de Início"
  3. No computador: Abra no Chrome → clique no ícone de instalar (⊕) na barra de endereço

Como atualizar o app depois

Atualizações automáticas

Sempre que você atualizar os arquivos no GitHub, a Vercel republica automaticamente em segundos. Basta:

  1. No GitHub, navegue até o arquivo que quer alterar
  2. Clique no ícone de lápis (editar)
  3. Faça a alteração
  4. Clique "Commit changes"
  5. A Vercel detecta e republica sozinha!

Para comercializar

Opções de monetização

  • Freemium: App gratuito com funcionalidades premium (ex: exportar relatórios, mais categorias)
  • Venda direta: Coloque o link na Hotmart/Kiwify como "acesso vitalício"
  • Assinatura: Adicione login (Firebase Auth) e controle de acesso por plano

Para a opção mais simples (venda direta), basta criar um produto na Hotmart com o link do app como "área de membros".
